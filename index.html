프롬프트 가이드 실험 · UI 미리보기

이 페이지는 연구자용 미리보기입니다. 실제 설문 참가자는 세 조건 중 하나(가이드 없음 / 가이드 A / 가이드 B)에 무작위로 배정되어, 그 조건에 맞는 화면과 구글폼만 보게 됩니다.

1️⃣ 실험용 프롬프트 학습 UI 템플릿

각 조건별로 목표 이미지 미리보기 → (필요 시) 가이드 → 프롬프트 실습 → “실행하기” 버튼 → 결과 미리보기 순서로 구성됩니다. 지금은 UI 구조만 확인하는 단계라 버튼은 실제로 이미지를 생성하지 않습니다.

CONDITION 1 · CONTROL

화면 1 · 프롬프트 가이드 없이 자유롭게 작성

1) 목표 이미지 안내

아래와 같은 느낌의 이미지를 만드는 것이 목표입니다. 똑같이 만들 필요는 없고, “책상 위에 펼쳐진 미니어처 공원”이라는 분위기만 비슷하면 충분합니다.

※ 실제 설문에서는 한 장 또는 두 장 정도만 보여주고 “참고용 예시일 뿐”이라고 안내하면 됩니다.

2) 프롬프트 작성

평소 사용하시던 AI 도구(챗GPT, Gemini 등)에 쓰듯이, 원하는 이미지를 만들 수 있도록 문장을 자유롭게 적어주세요.

이 영역 아래에 “첫 시도 / 수정 후 시도” 이미지를 나란히 보여주면, 참가자가 스스로 프롬프트를 고치면서 어떤 점이 달라지는지 체험할 수 있습니다.

첫 실행 결과 이미지 자리
수정 후 실행 결과 이미지 자리

CONDITION 2 · GUIDE A (BEGINNER)

화면 2 · 초보자용 단계별 프롬프트 가이드 + 실습

1) 목표 이미지 + 단계별 예시

아래 예시는 “느낌 → 무엇을 → 안에 뭐가 있는지 → 주변 → 품질” 순서로 적어본 것입니다. 그대로 따라 쓰지 않아도 되며, 구조만 참고하면 됩니다. 

미니어처 스타일로,
위에서 내려다본 각도의 책상 위 작은 공원을 만들어줘.

공원에는 작은 벤치, 미니 가로등, 나무와 꽃들,
산책하는 사람들, 뛰어노는 아이들, 작은 분수대가 있고,
배경에는 커피잔과 노트북이 흐릿하게 보이게 해줘.

밝은 아침 햇살 아래, 매우 상세하게, 4K 고화질로 표현해줘.
① 느낌(스타일/각도) ② 무엇을(대상) ③ 안에 있는 요소 ④ 주변/배경 ⑤ 품질/분위기

2) 가이드를 참고해서 직접 써보기

위 단계 ①~⑤를 모두 한 번씩 포함해 보면서, 자신만의 버전으로 문장을 작성해 보세요.

아래는 “가이드 적용 전/후”를 비교해서 보여줄 수 있는 자리입니다.

가이드 적용 전 이미지
가이드 적용 후 이미지

CONDITION 3 · GUIDE B (EXPERT)

화면 3 · 전문가용 프롬프트 가이드 + 실습

1) 목표 이미지 + 전문가용 예시

아래 예시는 구도, 이미지 요소, 디테일/품질을 의식해서 작성한 전문가용 샘플입니다. 

미니어처 디오라마 사진 스타일로,
45도 각도에서 내려다본 책상 위의 작은 공원을 생성해줘.

공원에는 미니 벤치, 작은 가로등, 나무와 화단, 작은 분수,
놀이터, 연못, 산책하는 사람과 뛰어노는 아이들이 등장하고,
배경에는 흐릿한 커피잔, 연필꽂이, 노트북이 배치되어 있게 해줘.

밝은 아침 햇살 아래, 매우 상세하고 사실적인 4K 고해상도로 표현하며,
미니어처의 스케일감과 깊이감을 특히 강조해줘.
  • 첫 블록 – 스타일(디오라마 사진), 시점(45도)을 고정
  • 둘째 블록 – 장면 구성 요소 + 사람의 활동을 구체화
  • 마지막 블록 – 빛·해상도·강조 포인트(스케일·깊이)를 지정

2) 자신의 스타일로 조정해 보기

평소 쓰시던 방식에 구도 / 요소 / 품질만 추가로 의식해서, 논문이나 수업 슬라이드에 그대로 넣어도 될 정도의 프롬프트를 한 번 작성해 보세요.

아래는 “가이드 사용 전에 본인이 쓰던 프롬프트 결과”와 “가이드 참고 후 수정한 결과”를 나란히 보여주는 영역입니다.

가이드 적용 전 (연구자가 저장한 이미지)
가이드 적용 후 (현재 프롬프트 결과)
